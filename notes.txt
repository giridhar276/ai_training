using venv library
----------------
step1: create virtual environment
        python3 -m venv myenv1 ( mac)
        python -m venv myenv1 ( windows)

step2: activate your virtual environment
      mac: source myenv1/bin/activate
      windows: .\myenv1\Scripts\activate

step3: you will start working/installing packages 
       if you want to crosscheck:
       windows:   where python 
       mac.   :   which python3
step4: deactivate 


Note: If you want the path of executable:
which python ( mac)
where python ( windows)
